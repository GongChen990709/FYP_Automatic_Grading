1. In the second input box - 'whether this function is overload or not'. If you choose 'yes', you can define multiple functions in a class with the same function name but different parameters (overload function). If you choose 'no', you can just define a function with unique function name.
2. After you set the number of parameters, you should click 'Confirm' button on the right and set the data type of each parameter in the drop-down box below.
3. When you have completed the setting of one overload function, you should firstly click 'Continue' button to store this function. Then if you want to continue to add overload function with the same function name as before, just set the 'Return Type' and 'the number of parameters', then click 'Continue' button to store this function. If you have finished the setting of these overload functions, click 'Add this function' button.
4. You can click "Click to generate a data type JSON file" to generate a json file and it can be sent to back-end of the system automatically.
